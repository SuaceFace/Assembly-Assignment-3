Instructions for this assignment:
Write a complete program that will input values for num1 ,num2, and num3 and display the value of the expression

        ( (num1 ^ 3) * num2 + 5 * ( num2 ^ 2) ) / num3.

        assume that the user enters only numbers that are greater than zero and the calculation never exceed 4 bytes size.

Sample run1:

num1 = 1
num2 = 2
num3 = 3
((num1 ^ 3) * num2 + 5 * ( num2 ^ 2)) / num3 = 7 R 1

sample run2:

num1 = 4
num2 = 6
num3 = 9
((num1^3)*num2 + 5*(num2^2)) / num3 = 62 R 6
